Routes:
- **POST /** : zValidator("json", createAgentSchema, zodErrorHook). Insert with db.insert(agents).values({...data, createdAt: now, updatedAt: now}).returning().get(). Catch UNIQUE constraint error -> 409. On success, call scheduleAgent(created, db). Return c.json(agent, 201).
- **GET /** : db.select().from(agents).all(). Return c.json(list).

Shared zodErrorHook function: maps result.error.issues to { field: issue.path.join("."), message: issue.message } details array, returns c.json({ error: "Validation failed", details }, 400).
