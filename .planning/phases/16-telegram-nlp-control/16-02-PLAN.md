Define the help text as a constant:
```typescript
const HELP_TEXT = `I can help you manage your agents. Try:

- "list agents" - show all agents
- "run [agent name]" - execute an agent
- "enable [agent name]" - enable an agent
- "disable [agent name]" - disable an agent
- "status" - system health summary
- "change [agent name] to [schedule]" - update schedule

Commands: /help, /start`;
```

**Helper: findAgentByName(name: string, db: Database): Agent | undefined**
- Query: `db.select().from(agents).where(sql\`${agents.name} COLLATE NOCASE = ${name} COLLATE NOCASE\`).get()`
- Return the agent or undefined

**Handler: handleHelp(): string**
- Return `HELP_TEXT`

**Handler: handleList(db: Database): string**
- Query all agents: `db.select().from(agents).all()`
- If no agents: return `"No agents configured."`
- For each agent, call `enrichAgent(agent, db)` to get enriched data
- Build response string with numbered list:
  ```
  Agents:
  1. Morning Briefing (enabled, healthy)
  2. PR Reminder (disabled, healthy)
  3. Error Monitor (enabled, unhealthy)
  ```
- Use status indicators: `"enabled"` / `"disabled"` for enabled flag, `"healthy"` / `"unhealthy"` for healthy flag

**Handler: handleRun(agentName: string, db: Database): string**
- Call `findAgentByName(agentName, db)`
- If not found: return `Agent "${agentName}" not found. Try: list agents`
- Fire-and-forget: `void executeAgent(agent, db).catch(err => console.error(\`[telegram-bot] Run "${agent.name}" failed: ${err instanceof Error ? err.message : err}\`))`
- Return `Running ${agent.name}...`
- NOTE: Do NOT check `agent.enabled` -- disabled agents can still be manually executed (per Phase 6 decision, POST /:id/execute works on disabled agents)

**Handler: handleEnable(agentName: string, db: Database): string**
- Call `findAgentByName(agentName, db)`
- If not found: return `Agent "${agentName}" not found. Try: list agents`
- If already enabled (agent.enabled === 1): return `${agent.name} is already enabled.`
- Update DB: `db.update(agents).set({ enabled: 1, updatedAt: new Date().toISOString() }).where(eq(agents.id, agent.id)).run()`
- Re-read agent from DB: `const updated = db.select().from(agents).where(eq(agents.id, agent.id)).get()!`
- Call `scheduleAgent(updated, db)` to register cron job
- Return `Enabled ${agent.name}.`

**Handler: handleDisable(agentName: string, db: Database): string**
- Call `findAgentByName(agentName, db)`
- If not found: return `Agent "${agentName}" not found. Try: list agents`
- If already disabled (agent.enabled === 0): return `${agent.name} is already disabled.`
- Update DB: `db.update(agents).set({ enabled: 0, updatedAt: new Date().toISOString() }).where(eq(agents.id, agent.id)).run()`
- Call `removeAgent(agent.id)` to unregister cron job
- Return `Disabled ${agent.name}.`

**Handler: handleStatus(db: Database): string**
- Query all agents: `db.select().from(agents).all()`
- Count enabled/disabled: `const enabledCount = allAgents.filter(a => a.enabled === 1).length`
- Count healthy/unhealthy using `getConsecutiveFailures`:
  ```typescript
  let unhealthyCount = 0;
  for (const agent of allAgents) {
    if (getConsecutiveFailures(agent.id, db) >= 3) unhealthyCount++;
  }
  ```
- Get circuit breaker status: `const cb = getLlmCircuitStatus()`
- Get scheduled jobs count: `const jobCount = getScheduledJobs().size`
- Build response:
  ```
  System Status: OK (or Degraded/Unhealthy)
  Agents: 5 total, 4 enabled, 1 disabled
  Health: 4 healthy, 1 unhealthy
  Scheduled jobs: 4
  LLM circuit breaker: CLOSED
  ```
- Status logic: if circuit breaker is OPEN -> "Unhealthy"; else if unhealthyCount > allAgents.length / 2 -> "Unhealthy"; else if unhealthyCount > 0 -> "Degraded"; else "OK"

**Handler: handleReschedule(agentName: string, scheduleInput: string, db: Database): Promise<string>**
- Call `findAgentByName(agentName, db)`
- If not found: return `Agent "${agentName}" not found. Try: list agents`
- Call `await parseSchedule(scheduleInput)`
- On success:
  - Update DB: `db.update(agents).set({ cronSchedule: result.cronExpression, updatedAt: new Date().toISOString() }).where(eq(agents.id, agent.id)).run()`
  - If agent is enabled (agent.enabled === 1):
    - Re-read from DB: `const updated = db.select().from(agents).where(eq(agents.id, agent.id)).get()!`
    - Call `scheduleAgent(updated, db)` to update cron job
  - Return `Updated ${agent.name} schedule to "${result.humanReadable}" (${result.cronExpression}).`
- On error: return `Could not parse schedule: ${err instanceof Error ? err.message : err}. Try a different description.`

**Handler: handleUnknown(): string**
- Return `I didn't understand that. Here's what I can do:\n\n${HELP_TEXT}`

**Main export: handleTelegramMessage(message: TelegramMessage, db: Database): Promise<void>**
- Extract `const text = message.text?.trim() ?? ""`
- Extract `const chatId = String(message.chat.id)`
- Get botToken from env: `const botToken = env.TELEGRAM_BOT_TOKEN!`
- Check for slash commands FIRST (no LLM call):
  - `if (text.toLowerCase() === "/start" || text.toLowerCase() === "/help")`:
    - `await sendPlainText(botToken, chatId, handleHelp())`
    - `return`
- Send typing indicator for LLM-processed messages: `await sendTypingAction(botToken, chatId).catch(() => {})`
- Query all agent names for intent parsing: `const allAgents = db.select().from(agents).all()`, `const agentNames = allAgents.map(a => a.name)`
- Call intent parser: `let intent: TelegramIntent`
  - try: `intent = await parseIntent(text, agentNames)`
  - catch: `await sendPlainText(botToken, chatId, "Something went wrong processing your message. Try again or type /help."); return`
- Dispatch based on intent.action:
  - `"list"` -> `reply = handleList(db)`
  - `"run"` -> if intent.agentName is null: `reply = "Please specify which agent to run. Try: list agents"`, else `reply = handleRun(intent.agentName, db)`
  - `"enable"` -> if null: `reply = "Please specify which agent to enable. Try: list agents"`, else `reply = handleEnable(intent.agentName, db)`
  - `"disable"` -> if null: `reply = "Please specify which agent to disable. Try: list agents"`, else `reply = handleDisable(intent.agentName, db)`
  - `"status"` -> `reply = handleStatus(db)`
  - `"reschedule"` -> if intent.agentName is null or intent.scheduleInput is null: `reply = "Please specify agent and schedule. Example: change Morning Briefing to every weekday at 9am"`, else `reply = await handleReschedule(intent.agentName, intent.scheduleInput, db)`
  - `"unknown"` -> `reply = handleUnknown()`
- Wrap dispatch in try/catch: on error, `reply = \`Something went wrong: ${err instanceof Error ? err.message : err}. Try /help.\``
- `await sendPlainText(botToken, chatId, reply)`
