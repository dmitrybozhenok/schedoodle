    2. **buildTelegramMarkdown(agentName: string, executedAt: string, output: AgentOutput): string** --
       - Use escapeMdV2 (`esc` alias) for all user text content
       - Structure: `*{esc(agentName)}*\n{esc(timestamp)}\n\n*Summary*\n{esc(output.summary)}\n\n*Details*\n{esc(output.details)}`
       - If output.data exists, append: `\n\n*Data*\n\`\`\`\n{escapeMdV2CodeBlock(dataStr)}\n\`\`\``

       - Timestamp: `new Date(executedAt).toLocaleString()`

    3. **buildTelegramFailureMarkdown(agentName: string, executedAt: string, errorMsg: string): string** --
       - Structure: warning emoji + `*FAILED: {esc(agentName)}*\n{esc(timestamp)}\n\n*Error*\n{esc(errorMsg)}`
       - Same truncation rule at 3800 chars

    4. **sendViaTelegram(botToken: string, chatId: string, text: string): Promise<NotifyResult>** --
       - Call sendTelegramMessage(botToken, chatId, text)
       - If result.ok: return { status: "sent" }
       - If !result.ok: console.error and return { status: "failed", error: result.description }
       - Catch thrown errors: return { status: "failed", error: message }
       - Pattern matches existing sendViaSmtp/sendViaResend exactly

    5. **Export sendTelegramNotification(agentName, executedAt, output): Promise<NotifyResult>** --

       - Call sendViaTelegram with env vars and message
       - Wrap in try/catch returning { status: "failed" } on unexpected errors

    6. **Export sendTelegramFailureNotification(agentName, executedAt, errorMsg): Promise<NotifyResult>** --
